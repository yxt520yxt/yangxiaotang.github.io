use_bert

使用bert_serving之前需要先安装tensorflow

pip install tensorflow==1.12(需要1.10及以上的版本)
pip install bert-serving-server==1.9.1
pip install bert-serving-client==1.9.1

下载bert模型并解压

https://storage.googleapis.com/bert_models/2018_11_03/chinese_L-12_H-768_A-12.zip

启动bert_serving服务

bert-serving-start -model_dir D:\python\chinese_L-12_H-768_A-12\chinese_L-12_H-768_A-12 -num_worker=2

使用命令行来运行脚本，因为IDE运行会报错

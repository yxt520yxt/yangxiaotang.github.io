Git

git的几个区

  • workspace：工作区
  • Index/Stage: 暂存区
  • Repository：本地仓库
  • Remote：远程仓库

下载安装完git之后，添加用户名以及邮箱的配置

  • git config –global user.name “yangxiaotang”
  • git config –global user.email yangxiaotang@stary.ltd

windows跟linux的换行符不一致，可能导致开发不一致

  • git config –global core.autocrlf input
  • git config –global core.safecrlf warn

上线时间跟当前不一致，创建新的分支

git fetch 拉取最新的信息
git checkout origin/master 切换分支
git checkout -f来撤销自己的修改
git checkout -b v1.1 = git branch v1.1 + git checkout v1.1(在切换后的分支创建分支)

查看本地分支对应关联的远端分支
git branch -vv

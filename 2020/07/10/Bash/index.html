Bash

shell脚本的第一行使用#！/bin/bash 指定解释器

运行shell脚本

  • 作为可执行程序运行，需要先赋予脚本可执行权限

    chmod +x shell_file
    ./yxt.sh

  • 作为解释器参数

    /bin/bash/ yxt.sh

shell变量

  • 变量名跟=不能有空格
  • 定义变量的时候，不需要$
  • 不可以使用标点符号，不能以数字开头，并且不能使用bash的关键字
  • 可以使用下划线
  • 变量可以重新定义，但是不可以重新赋值
  • readonly variable 变量设置为只读变量，不可重新定义
  • unset 删除变量，不可以删除只读变量
  • 没有强制使用{}包裹变量，规范最好就写上

三种变量

  1. 局部变量
  2. 环境变量
  3. shell变量

字符串

定义字符串的内容

  1. 单引号
  2. 双引号
  3. 或者两个都不要

    双引号可以做到转义以及内嵌变量，单引号不行

    字符串拼接

  • 使用双引号基本不需要拼接，变量直接写进去就行了
  • 单引号的拼接,也不需要+，直接拼起来就行

    字符串的长度获取

    ${ #variable_name}(#前的空格是为了越过hexo的错误)

    截取字符串

    ${variable_name:start_index:length}

    查找字符串的字符位置：（下标从1开始）

    expr index "${string}" ed

    匹配ed其中最先出现的字母

数组

定义（下标0开始，可以使用不连续的下标，并且下标的范围没有限制）

  1. array_name=(value1 value2 value3 …)
    • array[0]=value1
    • array[1]=value2

访问数组

  1. echo ${array_name} 输出的其实是array的第一个元素
  2. echo ${array_name[@]} 输出的才是整一个数组 *号跟@一样
  3. echo ${array_name[0]} 输出第一个元素
  4. echo ${ #array_name[@]} 计算数组长度(#前的空格是为了越过hexo的错误)
  5. echo ${ #array_name[1]} 计算数组第一个元素的长度(#前的空格是为了越过hexo的错误)

多行注释

参数传递

  • $0表示执行文件本身 $1 表示第一个命令行参数 …以此类推
  1. $* and $@ 获取所有的命令行参数（$*单字符形式, $@是多个）
  2. $# 获取命令行传递的参数个数
  3. $$ 脚本运行的当前进程id号
  4. $! 后台运行的最后一个进程ID
  5. $? 最后命令的退出状态，0表示没错，其他值均有错
  6. $- 当前所使用的选项

运算符

使用awk or expr来实现算数运算

  • example: echo expr $a + $b(需要转义，要\才可以)
  • 表达式运算符之间需要空格，没有就报错
  • 条件表达式的空格还要再多两个：[ $a == $b ](多了前后两个空格)

关系运算符

  1. -eq 检测两个数是否相等，相等返回 true。 $a -eq $b 返回 false。
  2. -ne 检测两个数是否不相等，不相等返回 true。 [ $a -ne $b ] 返回 true。
  3. -gt 检测左边的数是否大于右边的，如果是，则返回 true。 [ $a -gt $b ] 返回 false。
  4. -lt 检测左边的数是否小于右边的，如果是，则返回 true。 [ $a -lt $b ] 返回 true。
  5. -ge 检测左边的数是否大于等于右边的，如果是，则返回 true。 [ $a -ge $b ] 返回 false。
  6. -le 检测左边的数是否小于等于右边的，如果是，则返回 true。 [ $a -le $b ] 返回 true。

布尔运算符

  • ! 非运算，表达式为 true 则返回 false，否则返回 true。 [ ! false ] 返回 true。
  • -o 或运算，有一个表达式为 true 则返回 true。 [ $a -lt 20 -o $b -gt 100 ] 返回 true。
  • -a 与运算，两个表达式都为 true 才返回 true。 [ $a -lt 20 -a $b -gt 100 ] 返回 false。

逻辑运算符

  • && 逻辑的 AND [[ $a -lt 100 && $b -gt 100 ]] 返回 false
  • || 逻辑的 OR [[ $a -lt 100 || $b -gt 100 ]] 返回 true

字符串运算符

符号 描述 示例
= 检测两个字符串是否相等，相等返回 true。 [ $a = $b ] 返回 false。
!= 检测两个字符串是否相等，不相等返回 true。 [ $a != $b ] 返回 true。
-z 检测字符串长度是否为0，为0返回 true。 [ -z $a ] 返回 false。
-n 检测字符串长度是否不为 0，不为 0 返回 true。 [ -n “$a” ] 返回 true。
$ 检测字符串是否为空，不为空返回 true。 [ $a ] 返回 true。

read 脚本等待输入

  • -p 输入提示文字
  • -n 输入字符长度限制(达到6位，自动结束)
  • -t 输入限时
  • -s 隐藏输入内容
    1. 示例：read -p “you like me? ha?” -n 6 -t 3 -s variable_name
    2. 输入的词组以空格分开，如果词数大于变量数，就会将多出来的词组作为整体给最后一个参数接受

重定向

  1. 重定向输出到某个位置，替换原有文件的所有内容。

  2. 重定向追加到某个位置，在原有文件的末尾添加内容。

  3. < 重定向输入某个位置文件。
  4. 2> 重定向错误输出。
  5. 2>> 重定向错误追加输出到文件末尾。
  6. &> 混合输出错误的和正确的都输出。
